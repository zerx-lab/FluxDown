# Changelog

## v0.1.49 — 2026-07-03

### 🚀 新功能

- **本机 HTTP API 服务**：新增本地 API 服务（默认端口 17800，仅监听 127.0.0.1），
  支持浏览器脚本接管、aria2 JSON-RPC 兼容接口（addUri/getVersion/multicall 等）
  与带 token 鉴权的管理 API（任务增删查、暂停/恢复、队列查询），
  并提供 OpenAPI 3.1 规范与官网 `/api-docs` 在线文档
- **eD2K 协议支持**：新增 eD2K 链接下载，含服务器连接、KAD 网络与 UPnP 端口映射
- **桌面悬浮球**：新增可拖拽的桌面悬浮球，支持拖放链接投递下载与实时进度展示
- **阻止系统睡眠**：下载进行期间可阻止系统睡眠/息屏，避免下载中断
- **设置搜索**：设置页支持搜索直达并高亮目标设置项
- **自定义应用图标**：支持自定义窗口/任务栏/托盘图标
- **通知渲染升级**：Windows 悬浮通知改由 Flutter 离屏光栅化渲染，与应用主题/字体一致
- **右键菜单禁用态**：菜单项支持按上下文置灰禁用

### 🐛 修复

- 修复 macOS 托盘/Dock 点击无法恢复主窗口的问题（#420）
- HLS remux 与 DASH mux 前预检磁盘空间，避免空间不足导致合并失败
- 加固 BT 完成期文件搬运救援逻辑与 completion 幂等性
- 修复浏览器扩展对乱码及百分号编码 Content-Disposition 文件名的解码

### ⚡ 性能

- 浏览器扩展下载入口新增 NMH warmup 预热，App 冷启动与 cookie 收集并行；
  管道轮询改为固定间隔，降低首次投递延迟

### 🚜 重构与工程

- 下载引擎抽离为独立 `fluxdown_engine` crate，与 FFI 层（Rinf）完全解耦
- Release 流水线按组件拆分：桌面客户端、浏览器扩展、官网独立发布，
  按 tag 范围目录变更自动跳过未改动组件的构建
- 浏览器扩展新增 Edge Add-ons 商店自动发布
- 新增 AGPL-3.0 许可证与商标声明，中英双语 README
